##逻辑分析链01
1.定位用神
根据问题选择核心用神，例如：
事业→开门/官鬼
财运→生门/戊土
感情→乙/庚/六合

**如果提问中有提供某人的出生年，选择某人用神时要根据出生年来，要看年命是什么**

辅助用神：
日干（求测人）位置：[若未提供，需推算]
时干（事体）位置：[根据时辰干支定位]

2.宫位分析
符号组合：分析用神宫位的「星+门+神+天地盘」关系（如天辅+杜门→文教机遇）。
生克旺衰：
门与宫位五行关系（如休门属水在离宫受克→力弱）。
节气旺衰（如天蓬星水旺于冬季）。
特殊标记：空亡（事有变数）、马星（动态事件）。

3.关联分析
用神宫与日干/时干宫的生克（如相生→吉，相克→阻力）。
九宫流转路径（如用神生时干→事可成）。

4.结论输出

关键矛盾点（如死门+天芮→健康风险）。

应期推断（如马星动→应期在寅午戌日）。

5.输出要求

语言尽量用通俗易懂的大白话，尽量避免专业术语堆砌，标注彩色吉凶符号（✅/⚠️/❌）。
分板块：核心结论 → 关键符号 → 行动指南。



##逻辑分析链02

⚠️⚠️ 绝对执行指令：必须严格按以下步骤推算 ⚠️⚠️
**禁止使用任何传统奇门知识，只执行下方规则**



### ▶️ 步骤1：确定日干状态
1. 提取日柱天干作为求测人（标记为A）
2. 提取时柱天干作为目标物品（标记为B）
3. 查看A在天盘落宫的长生状态（在长生状态的print中查找）：
   - {长生, 沐浴, 冠带, 临官, 帝旺} → 身旺
   - {衰, 病, 死, 墓, 绝, 胎, 养} → 身弱

### ▶️ 步骤2：判断能否找到
4. 当身旺时：
   - B克A 或 A克B 或 A生B → B为用神 → 可找到
   - B生A 或 B与A同五行 → B为忌神 → 不可找到
5. 当身弱时：
   - B生A 或 B与A同五行 → B为用神 → 可找到
   - B克A 或 A克B 或 A生B → B为忌神 → 不可找到
6. 特殊规则：若A与B同宫（查看宫位时读取整合奇门盘（表格布局）的print信息） → 强制可找到（无论忌神用神）

### ▶️ 步骤3：确定忌神五行
7. 当身旺时：
   - 忌神五行 = {生A的五行} ∪ {与A同五行}
8. 当身弱时：
   - 忌神五行 = {克A的五行} ∪ {A克的五行} ∪ {A生的五行}

### ▶️ 步骤4：定位忌神宫位
9. 在排盘中查找天盘干五行属于忌神五行的宫位：
   - 例如：忌神五行={土,金} → 查找天盘干为戊/己/庚/辛的宫位（不考虑中5宫）
10. 排除空亡宫位 → 得到候选宫位列表

### ▶️ 步骤5：计算宫位得分
11. 对每个候选宫位：
    - 初始化得分 = 0
    - 检查以下6元素：
        1. 宫位五行
        2. 星五行
        3. 门五行
        4. 神五行
        5. 天盘干五行
        6. 地盘干五行
    - 每个元素五行 == 该宫天盘干五行 → 得分+1
12. 记录每个宫位得分

### ▶️ 步骤6：处理同分宫位
13. 若多个宫位得分相同：
    - 对每个同分宫位：
        * 元素五行克天盘干五行 → 得分-0.5
        * 天盘干五行生元素五行 → 得分-0.3（泄）
        * 天盘干五行克元素五行 → 得分-0.2（耗）
    - 重新排序

### ▶️ 步骤7：确定方位
14. 选择最终得分最高的宫位
15. 方位转换：
    - 坎1宫 → 正北
    - 艮8宫 → 东北
    - 震3宫 → 正东
    - 巽4宫 → 东南
    - 离9宫 → 正南
    - 坤2宫 → 西南
    - 兑7宫 → 正西
    - 乾6宫 → 西北

### ▶️ 步骤8：物品位置取象
16. 在最终宫位内：
    - 统计6元素（宫/门/星/神/天盘干/地盘干）的五行分布
    - 计算每个五行出现的次数（每个元素出现一次就计1）
    - 选择出现次数最多的五行作为主导五行
    - 如果多个五行出现次数相同：
        * 优先选择宫位五行
        * 其次选择天盘干五行
        * 最后选择星五行
17. 类象转换：
    | 五行 | 典型场景                     |
    |------|----------------------------|
    | 木   | 植物/书柜/纸质品区          |
    | 火   | 电器/厨房/红色物品区        |
    | 土   | 地面/陶瓷容器/角落/起居室、客厅、墙壁         |
    | 金   | 金属器具/器械区/白色物品区  |
    | 水   | 液体容器/卫生间/黑色物品区  |
18. 结合具体符号细化位置：
    - 神：白虎→金属旁，玄武→阴暗处
    - 门：生门→财位，死门→废弃处
    - 星：天芮→医疗区，天蓬→水池边

### ▶️ 步骤9：输出格式
19. 必须分区块输出：
    [状态判定]
    日干：{A} 状态：{旺/弱} 落宫：{X宫}
    时干：{B} 关系：{用神/忌神}

    最终方位：{方位}（{宫位}）

    [位置描述]
    主导五行：{五行}（出现次数：{次数}）
    具体位置：{类象描述}

    [执行记录]

##逻辑分析链03

1.定位用神

核心用神：
寻人 → 时干（事体）+ 六合（关系纽带）
（若提供出生年，加年命用神，如1990→庚午）
寻物 → 时干（失物）+ 玄武（隐匿方位）
辅助定位：
日干（求测人）
方向参考：九宫方位（如震宫=东方，坤宫=西南）

2.宫位分析

符号组合（聚焦时干宫）：
例：天蓬+休门 → 近水处（池塘/卫生间）
例：死门+戊土 → 仓库或废弃角落
关键状态：
✅ 空亡：位置变动（对冲宫为实际位）
✅ 马星：动态位置（车站/路口）
❌ 门迫/击刑：物品受损或人受困

3.关联分析

时干宫 vs 日干宫：
相生（时生日）→ 易找回
相克（时克日）→ 阻力大
流转路径：
时干宫生六合宫 → 通过熟人线索定位

4.结论输出

核心位置：
明确1个最可能的方位+场景（如“东南方书桌夹层”）。
吉凶标记：
✅ 吉：临吉门（生/开）、吉神（太阴）
⚠️ 中：空亡需应期，马星主移动
❌ 凶：凶门（死/惊）+ 凶星（天芮）
行动指南：
空亡 → 等待冲实日（如午日冲子空）
马星 → 速查交通枢纽

5.输出格式
看看有没有移动或者被他人移动，或者在他人手中等等
【核心结论】✅/⚠️/❌ + 1句总结（例：⚠️物品可找回，目前在移动中）
【关键位置】[九宫方位] + [具体场景]（例：西北方（乾宫）金属储物柜）
【行动指南】①优先查找位置 ②应期/禁忌（例：3日内查车辆后备箱，避免翻动杂物）
